# Security & Data Protection

## Our Commitment to Security

PressClass AI is built with security as a foundational requirement, not an afterthought. We recognize that educational data is sensitive by nature, and we apply industry-standard technical and organizational safeguards to protect user information from unauthorized access, loss, or misuse.

---

## Data Encryption

### Data in Transit

All data transmitted between your device and PressClass servers is encrypted using **Transport Layer Security (TLS) version 1.2 or higher**. This ensures that information remains confidential and protected from interception during transmission.

### Data at Rest

Data stored on PressClass servers is encrypted using **AES-256 encryption standards**, which are widely recognized and used for securing sensitive information across industries.

## Access Control

PressClass AI employs **strict role-based access control (RBAC)** to limit internal access to systems and data.

- Only authorized personnel can access backend systems  
- Access is granted strictly on a need-to-know basis  
- User-generated content is not viewed by staff unless explicit permission is granted for support or troubleshooting purposes

Administrative access is logged and monitored to ensure accountability.
